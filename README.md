# Silent-Auction-World-Cup-2026-Pot-2-Sweepstake
An interactive sweepstake for Pot 2 of the World Cup 2026 teams. Set up for a fun way with my mates. For 2 - 6 players. Can make it up to 12 easily though

The API Key is on show. That's intentional and fine.
Firebase API keys are not secrets, they're identifiers. Security is enforced in two layers: the key is restricted to requests coming from this domain only, so it's useless anywhere else; and the database requires a valid authenticated session before it will accept any read or write. There's no sensitive data here anyway. It's a sweepstake
